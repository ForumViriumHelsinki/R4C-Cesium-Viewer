## Project Lifecycle

Projects move through these phases:

1. **Planning** - PRD created, requirements defined
2. **Implementation** - Active development with phase tracking
3. **Testing** - Comprehensive test coverage added
4. **Completed** - Project finalized, documentation archived if needed

## Creating a New Project

When starting a new project:

1. Create a new subdirectory: `docs/projects/project-name/`
2. Add a PRD.md with requirements and scope
3. Track implementation progress with phase documents
4. Document test coverage and deliverables
5. Create a README.md overview
