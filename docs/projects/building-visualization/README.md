# Building Visualization Project

3D building rendering, data display, and interaction features for the R4C Cesium Viewer.

## Status

**Planning Phase**

## Documents

- [PRD.md](./PRD.md) - Product Requirements Document

## Project Scope

The Building Visualization project focuses on:

- 3D building models and rendering
- Building attribute data display
- Energy efficiency visualization
- Building selection and highlighting
- Detail panels and information overlays

## Related Components

- `src/pages/Building.vue` - Building detail view
- `src/services/building.js` - Building data service
- `src/stores/buildingStore.js` - Building state management
- `src/stores/propsStore.js` - Property and attribute data

## Data Sources

- HSY Buildings dataset
- Energy performance certificates
- Building ownership and property data
- 3D geometry from Helsinki Region Environmental Services
