This folder contains documentation for the data models and GeoJSON structures used in R4C-Cesium-Viewer.

| -------------------------------------------------- | ----------------------------------------------------------- |
| [BUILDING_PROPERTIES.md](./BUILDING_PROPERTIES.md) | Comprehensive reference for all building GeoJSON properties |

## Quick Reference

### Building Data Sources

- **Capital Region View**: HSY (Helsinki Region Environmental Services) data with heat timeseries
- **Helsinki View**: City of Helsinki data with SOTE classifications and detailed building info
- **Mock API**: Synthetic data generated by `mock-api/generate.ts` for development

### Key Property Groups

1. **Identifiers**: `vtj_prt`, `ratu`, `id`, `hki_id`
2. **Location**: `postinumero`, `kunta`
3. **Building Type**: `kayttarks` (string), `c_kayttark` (numeric code)
4. **Physical**: `i_kerrlkm`, `measured_height`, `area_m2`, `rakennusaine_s`
5. **Heat Data**: `heat_timeseries`, `avgheatexposuretobuilding`, `distancetounder40`
6. **Population**: `pop_d_0_9` through `pop_d_over80`

See [BUILDING_PROPERTIES.md](./BUILDING_PROPERTIES.md) for complete details.
